
# Pintos Operating System Project

## Overview

> For a Chinese version of this document, see README.zh-CN.md

This project extends the Pintos teaching operating system with
thread scheduling, user process support, and a file system
implementation based on FFS. The focus is on correctness and
synchronization under concurrent workloads.
